How to Play

  • Normal state (gray face): Run as far as possible jumping abstacles. Running far or eating hot herb makes you heated state
  • Heated state (red face): Eat 3 Cool herbs to return to normal
  • Game Over: Eating hot herb in normal state, or Eating cool herb in normal state
  • Controls: Space/Click/Touch to jump
  • 평상시 (회색 얼굴): 장애물을 피하며 가능한 멀리 달리세요. 일정시간 달리거나 더운 약재를 먹으면 열난 상태가 됩니다.
  • 열이 날 때 (빨간 얼굴): 차가운 약재 3개를 먹어 정상으로 돌아오세요
  • 게임오버: 보통 상태에서 찬약을 먹거나, 열난 상태에서 더운 약을 먹는 경우
  • 조작: 스페이스/클릭/터치로 점프
