변증 기초

🧭팔강 (八綱)

  • 한(寒) - 추위, 느림(→기체,혈어), 지맥, 맑은 물(콧물, 소변, 가래), 설사, 득온즉감
  • 열(熱) - 더위, 번조, 삭맥, 누런 물, 변비, 득한즉감
  • 표(表) - 주로 급성병, 피부 (* 나중에 가면 급성병만 표병이라고 함)
  • 리(裏) - 주로 만성병, 내장
  • 허(虛) - 좋은 것이 부족한 것(정기, 혈, 정)
  • 실(實) - 나쁜 것이 많은 것 (한, 습, 기체, 어혈, 담, etc)
    * 간기가 너무 실해서 병이되는가? - No, 간의 실증은 간기체, 간혈허 등등 (오행침은 별개)
  • 음허(陰虛) - 조열, 오장심열, 야간심(모든 음부족), 식은 땀, 세맥
  • 양허(陽虛) - (기허) - 추위, 부종, 주간심, 자한, 허맥

    * 음에 관한 것은 밤, 오후 심, 양은 아침, 낮 (땀, 열, 통증)

    * 양기 허 → 음기 실 (한, 습) - 추위, 부종 / 음기 허 → 양기 실 (조, 열) - 건조, 열증

    * 양실(고열) → 음기 허(건조) / 음실(추위,습) → 양기 허

💧기, 혈, 정 (氣血精)

기(氣)에 관한 것

  • 기허(氣虛) - 무력 / 폐-기단 / 비 - 식욕
  • 기역/기함 - 구토, 기침 / 장기하수
  • 기체(氣滯) - 통증 - 움직임, 실체없음, 팽만 cf.혈어

혈(血)에 관한 것

  • 혈어(血瘀) - 자통, 실체있음, 고정, 야간심, 삽맥, 모든 보라색 징후
  • 혈허(血虛) - 창백(pale, sallow), 손발참, 세맥

정(精)에 관한 것

  • 정허(精虛) - 발육부진, 성기능, 조기 노화
* 신 따로 변증이 별로 없다. 진액부족은 대체로 음허에 해당

💨육기 (六氣)

  • 풍(風) - 빠름(급성병은 보통 풍, 감기), 떨림, 근육 쥐나는 것, 경련(마비)
  • 습(濕) - 무거움, 양허와 관련
* 한열은 팔강, 조는 음허와 일치, 서는 임상에서 다루지 않음

🫀오장 생리 (五臟 生理)

  • 간(肝) - 혈을 저장, 감정중 용맹, 결단, 근육(운동, 건)을 주관, 옆구리, 눈
  • 심(心) - 혈액순환기능, 정신활동의 전체를 통솔, 혀
  • 비(脾) - 소화기능, 기육(살,근육)을 주관함, 후천적인 에너지 공급
  • 폐(肺) - 호흡기능, 면역기능
  • 신(腎) - 비뇨/생식기, 우리 몸의 가장 근원 명문, 정을 주관, 귀

🩺오장 변증 (五臟 辨證)

🌿간변증

  • 간기울결 - 정서문제. 협륵, 월경문제, 현맥 + 기체
    * 기체(통증,팽만)+ 간(월경, 근육, 옆구리) = 협륵통, 월경통, 근육통
  • 간화상염 - 분노, 충혈, 측두통 + 열증
  • 간풍 - 근육경련, 반신마비, 뇌졸중
  • 간혈허 - 현기증(이명), 시야 흐림, 약한 손톱, 근육 약화와 경련 + 혈허

❤️심변증

  • 심화상염 - 심계, 불안, 혀의 궤양, 진한 소변, 불면(흥분) + 열증
  • 심혈허 - 심계, 불안 + 혈허(창백, 손발참)
  • 심혈어 - 심계, 불안, 심장부위 조이는 통증 + 어혈(자통, 실체있음, 고정, 야간심)

🌾비위변증

  • 비기허 - 창백한(sallow) 안색, 식욕 부진+ 기허
  • 비양허 - 비기허 + 추위, 설사, 무거움, 부종
  • 비기하함 - 비기허 + 장기하수
  • 비불통혈 - 비기허 + 출혈
  • 위음허 - 상복부 공복통, 속쓰림, 구강건조, 구내염, 갈증 + 음허(오후열)
  • 식체위완 - 구토, 입 냄새, 트림, 위완통
* 위는 비의 일부로 생각하면 편함, 다른 육부는 중요하지 않음

🌬️폐변증

  • 폐기허 - 숨가쁨, 기침. 주간 발한, 면역약화 + 기허
  • 폐음허 - 마른 기침(가래가 없음), 목의 건조 + 음허
  • 폐실증 - 풍한범폐, 풍열조폐 - 감기 (풍,열,한)

🌊신변증

  • 신양허 - 요통, 무릎, 이명, 성기능 문제 + 양허(부종, 추위), 맑은 소변
  • 신음허 - 요통, 무릎, 이명, 성기능 문제 + 음허
  • 신정허 - 요통, 무릎, 이명, 성기능 문제 + 어린이: 발달장애, 성인: 노화
